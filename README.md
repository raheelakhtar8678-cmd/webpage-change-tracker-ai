# Webpage Change Tracker + AI Summary

Monitor any webpage and detect content changes over time.
Optionally generate an AI explanation describing what changed and why it matters.

## Features
- Text-based change detection
- Highlights added and removed content
- Optional AI-generated explanation
- Supports OpenAI, Gemini, OpenRouter
- Works with Apify scheduling

## Use Cases
- Monitor pricing pages
- Track policy updates
- Watch product announcements
- Monitor blogs or news pages

## Input
- Webpage URL
- Optional AI provider and API key

## Output
- Change detection result
- List of changes
- AI summary (if enabled)

## Tip
Schedule this Actor daily or weekly to continuously monitor changes.
